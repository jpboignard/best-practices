## Description

Use CSS shorthand notations to reduce stylesheet size.

## Example

Do not write :

```css
margin - top: 1em; margin - right: 0; margin - bottom: 2em; margin - left: 0.5em;
```

But rather :

```css
margin: 1em 0 2em 0.5em;
```

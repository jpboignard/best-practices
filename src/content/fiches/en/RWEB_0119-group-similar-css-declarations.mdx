## Description

When multiple DOM (Document Object Model) elements share common CSS properties, declare them together in the same style sheet. This method reduces the weight of CSS.

## Exemple

Don't write : \\

```css
h1 {
  background - color: gray; color: navy;
}

h2 {
  background - color: gray; color: navy;
}

h3 {
  background - color: gray; color: navy;
}
```

But rather :

```css
h1, h2, h3 {
  background - color: gray; color: navy;
}
```

This rule can be enforced with a linter.
